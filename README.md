# Neumorphism Custom Checkbox with HTML & CSS | CodingNepal

Um projeto simples de **checkbox personalizado com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **checkbox moderno com estilo neumorphism**, utilizando sombras suaves para um visual elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Checkbox estilizado
- Efeito neumorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
